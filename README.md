# Social Media Web Application (Django)

A full-stack social media platform built using Django that enables users to connect, interact in groups, and share content efficiently.

## Features

- User authentication (signup, login, logout)
- Create, edit, and delete posts (CRUD operations)
- Group-based interactions (join and participate in groups)
- User tagging using @username
- Responsive UI for seamless experience across devices

## Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite / PostgreSQL (whichever you used)
- **ORM:** Django ORM
- **Architecture:** Class-Based Views (CBVs)

## Key Highlights

- Designed scalable backend using Django class-based views  
- Implemented relational database models for users, posts, and groups  
- Built dynamic user interactions including tagging and group participation  
- Followed modular and maintainable project structure  


## Future Improvements

- Add real-time chat using WebSockets (Django Channels)
- Implement notifications system
- Improve UI/UX with modern frontend frameworks
- Deploy on cloud (AWS / Render / Vercel)
